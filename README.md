# Enregistreur Protection Civile

Application macOS d'enregistrement d'écran à destination des bénévoles de la
Protection Civile.

## Téléchargement

Les versions stables sont publiées sur la page
[Releases](https://github.com/leomineur/enregistreurpc/releases) de ce repo.

La dernière version recommandée est toujours disponible via
[le tag `latest`](https://github.com/leomineur/enregistreurpc/releases/latest).

## Mises à jour automatiques

L'application se met à jour automatiquement via
[Sparkle](https://sparkle-project.org). Elle vérifie une fois par jour le
fichier [`appcast.xml`](./appcast.xml) à la racine de ce repo et propose
l'installation des nouvelles versions signées.

Pour vérifier manuellement : menu **Enregistreur Protection Civile →
Vérifier les mises à jour…**.
